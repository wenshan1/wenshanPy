# MYSQL 中文解决方案
## windows解决方案一 
设置MySQL数据库默认编码my.ini文件中，default-character-set=utf8   (注意是utf8中间没有-) [mysql] default-character-set=utf8 [mysqld] default-character-set=utf8

## Linux解决方案 
创建数据库是用utf8： create database blan charset utf8;
